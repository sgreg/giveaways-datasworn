# giveaways-datasworn

Datasworn YAML sources and compiled JSON files for the Discord giveaway assets.

>[!NOTE]
> This is a work in progress and has only been tested with [Iron Vault](https://ironvault.quest/)'s [Homebrew system](https://ironvault.quest/other-features/rulesets-and-homebrew.html)
